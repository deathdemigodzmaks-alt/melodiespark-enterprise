## 🚀 QUICK START (15 MINUTES TO LIVE)

### Step 1: Deploy Frontend to Netlify (5 min)

```
1. https://app.netlify.com/signup
2. Sign up with GitHub
3. Click: "Add new site" → "Import an existing project"
4. Select: melodiespark-enterprise
5. Build command: npm run build
6. Publish: apps/web/.next
7. Click: "Deploy site" ✅
```

### Step 2: Deploy Backend to Railway (5 min)

```
1. https://railway.app/signup
2. Sign up with GitHub
3. Click: "New Project"
4. Select: "Deploy from GitHub repo"
5. Choose: melodiespark-enterprise
6. Root: apps/api
7. Click: "Deploy" ✅
```

### Step 3: Add Databases (3 min)

```
In Railway:
1. Click: "New"
2. Select: "Database"
3. Choose: "PostgreSQL" ✅
4. Repeat for: "Redis" ✅
```

### Step 4: Set Variables (2 min)

```
In Netlify:
- NEXT_PUBLIC_API_URL = [your-railway-url]

In Railway:
- DATABASE_URL = [auto-filled]
- REDIS_URL = [auto-filled]
- JWT_SECRET = [generate random]
- API keys = [add yours]
```
