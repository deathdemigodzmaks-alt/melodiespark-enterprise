# Automation Configurations

All configuration files for automated deployment are stored here.

## Files

- `vercel-config.json` - Vercel deployment settings
- `railway-config.json` - Railway services configuration
- `github-env.yml` - GitHub Actions environment
- `.env.template` - Environment variables template
- `monitoring-config.json` - Health check & monitoring setup
- `state-schema.json` - Deployment state structure
- `rollback-config.json` - Automatic rollback settings
- `performance-tuning.json` - Performance optimization

## Usage

1. Update variables with your values
2. Copy to appropriate deployment targets
3. Run automation scripts
