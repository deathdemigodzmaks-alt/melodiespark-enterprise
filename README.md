# 🚀 MelodieSpark Enterprise SaaS Platform

## Quick Start

```bash
# Install dependencies
pnpm install

# Setup environment
cp .env.example .env

# Start development
pnpm dev

# Run tests
pnpm test

# Deploy
pnpm deploy:cloud
```
